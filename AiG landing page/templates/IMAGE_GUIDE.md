# Image Guide for AIESEC Landing Page

This document explains where to place your images for the landing page.

## Image Locations

All images should be placed in the `public` folder of your project.

## Required Images

### 1. Logo
- **Filename:** `logo.png`
- **Location:** `/public/logo.png`
- **Usage:** Navigation bar and footer
- **Recommended size:** 200x50px (transparent background)

### 2. Hero Section
- **Filename:** `hero-person.png`
- **Location:** `/public/hero-person.png`
- **Usage:** Walking person silhouette on the right side of hero section
- **Recommended size:** 600x600px (transparent background)

### 3. Cultural Diversity Circle Images (Introduction Section)
- **Filenames:** `cultural-diversity-1.png` through `cultural-diversity-8.png`
- **Location:** `/public/cultural-diversity-[1-8].png`
- **Usage:** Small circular images on left and right of introduction text
- **Recommended size:** 300x300px each

### 4. Product Icons
- **Filename:** `volunteer-icon.png`
  - Location: `/public/volunteer-icon.png`
  - Red-themed icon for Global Volunteer
  
- **Filename:** `talent-icon.png`
  - Location: `/public/talent-icon.png`
  - Cyan-themed icon for Global Talent
  
- **Filename:** `teacher-icon.png`
  - Location: `/public/teacher-icon.png`
  - Orange-themed icon for Global Teacher
  
- **Filename:** `member-icon.png`
  - Location: `/public/member-icon.png`
  - Blue-themed icon for AIESEC Member

- **Recommended size:** 200x200px (transparent background)

### 5. Testimonial Images
- **Filenames:** `testimonial-1.png` through `testimonial-4.png`
- **Location:** `/public/testimonial-[1-4].png`
- **Usage:** Profile images in testimonial cards
- **Recommended size:** 200x200px

### 6. Partner Logos
- **Filenames:** `partner-1.png` through `partner-22.png`
- **Location:** `/public/partner-[1-22].png`
- **Usage:** Partner company/organization logos
- **Recommended size:** 150x150px

## Quick Setup

1. Create/navigate to the `public` folder in your project root
2. Add all images with the exact filenames mentioned above
3. The images will be automatically loaded when you run the development server

## Color Scheme Reference

- **Primary Blue:** #0a7aff
- **Secondary Blue:** #2196f3
- **Red (Volunteer):** #ff6b6b
- **Cyan (Talent):** #17c3b2
- **Orange (Teacher):** #ff9f1c

## Notes

- All images in the `public` folder are accessible via `/filename.png` in your code
- Make sure image filenames match exactly (case-sensitive)
- Use PNG format for images that need transparency (logo, hero person, icons)
- Use JPG format for photos (testimonials, cultural diversity images)
